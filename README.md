<!-- GETTING STARTED -->
## Getting Started


### Prerequisites

* .NET Framework
  ```sh
  https://dotnet.microsoft.com/download/dotnet-framework/net472
  ```
* Visual Studio
  ```sh
  https://visualstudio.microsoft.com/
  ```

  
  
  
  

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/BeeBeeCue/Aktia_Csharp
   ```
2. Open in Visual Studio
3. Run without debugger
   ```sh
   Ctrl + F5
   ```
